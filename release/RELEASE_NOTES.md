Initial public release.

## Download

```text

```

Open the DMG and double-click `Install Codex CLI UI.command`.

- `Codex CLI UI.app` launcher with red rounded icon

- macOS
- Codex Desktop / Codex CLI
- Ollama
- `gpt-oss:20b`
