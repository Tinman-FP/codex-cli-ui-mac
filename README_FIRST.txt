Codex CLI UI for Mac

1. Open this DMG.
2. Double-click "Install Codex CLI UI.command".
3. The app will be installed to ~/Applications/Codex CLI UI.app.
4. The local UI will run at http://127.0.0.1:8765.

Requirements:
- Codex Desktop installed at /Applications/Codex.app
- Ollama installed
- gpt-oss:20b pulled with: ollama pull gpt-oss:20b

This package does not include private Codex chat history.
