# Security

## Supported Use

This package is intended for local Mac use with Codex Desktop, Codex CLI, and Ollama.

The UI binds to:

```text
127.0.0.1:8765
```

It is not intended to be exposed on a public network.

## Reporting Issues

Open a GitHub issue with:

- macOS version
- Codex CLI version
- Ollama version
- Installer output
- Relevant local UI logs from `~/Applications/Codex_CLI_UI/logs`

Do not paste secrets, private session transcripts, API keys, or VPN-only host details.
